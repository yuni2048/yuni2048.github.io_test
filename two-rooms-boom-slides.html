🔄 인질 교환

각 라운드가 종료되면 리더가 인질을 선정해 두 방이 서로 사람을 교환한다.

  • 1
    라운드 종료 시 각 방의 리더가 인질 명수를 확인한다.
  • 2
    리더는 인질을 지정하고, 두 방의 리더가 인질을 교환한다. 리더 본인은 인질이 될 수 없다.
  • 3
    최종 라운드 인질 교환 후, 모든 플레이어가 카드를 공개하고 승패를 결정한다.
인질 수 기준 (3라운드): 1라운드 — 방 인원의 1/4, 2라운드 — 방 인원의 1/4, 3라운드 — 1명
👑 리더 역할
  • 1
    게임 시작 시 각 방에서 투표로 리더를 선출한다. 가장 많은 지목을 받은 사람이 리더.
  • 2
    하야: 현재 리더가 자발적으로 다른 플레이어에게 리더 카드를 넘길 수 있다.
  • 3
    탄핵: 아무 플레이어나 탄핵을 제안 가능. 손들어 과반수 지목 시 리더 교체.
리더 교체는 라운드 중 언제든 시도할 수 있으나, 마지막 라운드는 주의가 필요하다.
🗣️ 대화 & 카드 공유

라운드 중에는 어떤 행동도 허용된다. 전체 대화, 밀담, 카드 공유 모두 가능하다.

🤝 카드 공유
1:1로 서로 카드를 동시에 보여준다. 기본 추천 방식.
🎨 색상 공유
카드 내용 없이 팀 색깔(블루/레드)만 공개한다.
🔐 개인 공개
특정 1명에게만 카드를 보여준다.
📢 전체 공개
방 전체에 자신의 카드를 공개한다.
⚠️ 주의: 거짓말, 카드 숨기기, 카드 일부만 보여주기 모두 허용. 단, 라운드 시작 직전에 카드를 미리 공개하는 것은 금지. 인질 교환 중(라운드 사이) 정보 공유 금지.

🔮 상태(Status) 설명

일부 캐릭터는 특정 상태(Status)를 부여하거나 보유한 채 시작한다. 상태는 행동 제한 또는 특수 규칙을 만들어낸다.

🤫 수줍음 (Shy)
카드의 어떠한 부분도 공개할 수 없다. 색상 공유, 카드 공유, 개인 공개, 전체 공개 모두 불가. 단, 심리학자가 먼저 개인 공개를 해오면 카드 공유 가능. 어리석음 상태와 동시 보유 시 두 상태 모두 소멸.
🔴 범죄자, 블루/레드 부끄럼쟁이
👻 저주 (Cursed)
최선을 다해 소리를 내지 않아야 한다. 언어 사용이 필요한 모든 능력(에이전트, 분노광, 좀비 전환 등)을 사용할 수 없게 된다.
🧟 미라 (블루/레드)
🙈 수줍음(Coy)
색상 공유만 가능. 카드 내용 공개, 카드 공유, 개인 공개 불가. 어리석음 상태와 동시 보유 시 두 상태 모두 소멸.
🤐 부끄럼쟁이 (코이보이), 깡패
😜 어리석음 (Foolish)
카드 공유나 색상 공유 제안을 절대 거부할 수 없다. 수줍음/코이 상태와 동시 보유 시 두 상태 모두 소멸.
🍀 레프리콘, 어리석은 자, 딜러
🛡️ 면역 (Immune)
예외 없이 모든 능력과 상태에 영향을 받지 않는다. 에이전트의 강제 공개도 무시 가능. 단, 스스로 카드 공유를 하는 것은 가능.
🛡️ 면역학자, 문지기 (대사)
😇 정직함 (Honest)
말로 하는 모든 것을 진실만 말해야 한다. 말이 아닌 행동으로는 속임 가능. 거짓말쟁이 상태와 동시 보유 시 두 상태 모두 소멸.
😇 천사 (Angel)
😈 거짓말쟁이 (Liar)
말로 하는 모든 것을 거짓만 말해야 한다. 말이 아닌 행동으로는 진실 표현 가능. 정직함 상태와 동시 보유 시 두 상태 모두 소멸.
😈 악마 (Demon)
💀 사망 (Dead)
폭발범과 같은 방에 있던 모든 플레이어가 얻는 상태. 대통령이 이 상태가 되면 레드팀 승리. 좀비 폭발범의 경우 모든 방에 적용.
💣 폭발범 (게임 종료 시)
🧟 좀비 (Zombie)
팀 좀비로 진영이 바뀐다. 카드/색상 공유를 하면 상대방도 좀비가 된다. 팀 좀비는 게임 종료 시 사망 상태가 아닌 모든 플레이어가 좀비일 때 승리.
🧟 좀비 캐릭터
🔗 현명함 (Savvy)
카드 공유만 가능. 색상 공유, 개인 공개, 전체 공개는 불가. 코이 상태가 추가로 생기면 모든 공개 방법이 막힌다.
🤝 경호원 (협상가), 교수
😵 최면 (Hypnotized)
최면술사가 지정한 캐릭터인 척 행동해야 한다. 실제 팀과 능력은 바뀌지 않으며, 단지 해당 캐릭터처럼 연기해야 한다.
😵 최면술사 (Hypnotist)
💘 사랑에 빠짐 (In Love)
원래 승리 조건을 잃고, 게임 종료 시 상대방과 같은 방에 있어야만 승리 가능. 증오 상태와 동시 보유 시 두 상태 모두 소멸.
